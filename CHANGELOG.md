## [1.0.0] — 2026-02-26

### Added

- AI-powered presentation generation from a topic with configurable slide count, language, and style
- Real-time streaming generation — watch slides build live
- 9 built-in themes with custom theme creation
- Full rich-text editing powered by Plate Editor
- Drag-and-drop slide reordering via DND Kit
- Local AI model support via Ollama and LM Studio (no API keys required)
- OpenAI and Together AI integration for cloud model support
- Google OAuth authentication via NextAuth.js
- Image generation with support for multiple AI providers
- Stock photo support via Unsplash
- Web-search-augmented generation via Tavily
- Export to PowerPoint (.pptx)
- Presentation mode for direct in-browser presenting
- Auto-save with optimistic updates
- Auto-design engine for consistent slide styling
- Brand kit management
- Template library foundation
- AI copilot and inline AI suggestions in the editor
- Version history / versioning service
- Analytics service
- Webhook support
- Rate limiting and security middleware
- PostgreSQL + Prisma ORM data layer
- WebSocket support (Yjs-based collaboration foundation)
- E2E tests with Playwright
- Unit tests with Vitest
- Full CI pipeline via GitHub Actions
